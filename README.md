# HRNet

## Requirements

You will need either [bun](https://bun.sh) v^1.1 or [node](https://nodejs.org) v^20.12

## Setup

```
git clone git@github.com:Zuruuh/hrnet.git
cd hrnet
```

### Start dev server with Bun

```
bun install
bun run dev
```

### Start dev server with node

```
npm install
npm run dev
```
